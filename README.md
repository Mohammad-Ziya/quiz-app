## How the App Works

- Quiz questions are stored as an array of objects in JavaScript.
- Each question and its options are rendered dynamically on the page.
- When a user selects an option:
  - The answer is checked against the correct one.
  - Correct and wrong options are highlighted.
- Score and progress update automatically.
- After the final question, the quiz ends and a restart button appears.
